Диапазон версий включительно
```xml
        <dependency>
            <groupId>test</groupId>
            <artifactId>test</artifactId>
            <version>[1.0.0 - 1.0.12]</version>
        </dependency>
```
Диапазон версий исключительно 
```xml
        <dependency>
            <groupId>test</groupId>
            <artifactId>test</artifactId>
            <version>(1.0.0 - 1.0.12)</version>
        </dependency>
```
